工作机会

在数鹏通（LinkCM）科技，我们追求做到最好，拼实力、拼努力，不拼爹；我们追求尊重平等，经理与员工一样的座位，平等开放，没有官僚；我们追求以人为本，对所有员工每年均有完善的职业生涯规划，所有规划均由总经理一一详读；我们追求进步与发展，积极拥抱高新科技的浪潮，从IT到互联网到移动互联网。我们，期待您的加入！

  • JAVA工程师 岗位职责：
    1. 在项目经理/技术经理的带领进行软件系统的开发，如规范、设计、编码、性能调试；
    2. 遵循公司的项目管理和软件开发流程,在规定时间内提交合乎要求的产品；
    3. 具有良好的沟通能力，以确保团队及成员能够明确共同目标，并能进行良好的协调和合作；
    专业能力要求：
    1. 本科学历，计算机及相关专业；
    2. 两年以上JAVA软件开发经验；
    3. 精通struts + spring + hibernate等开源软件；
    4. 精通JavaScript；
    5. 熟悉Tomcat，Ant和CVS；
    6. 了解数据库原理，SQL语言，和主流数据库Oracle, Sql Server, MySQL；
    7. 了解软件架构，能够在多层架构系统中独立开发和集成组件；
    8. 具有良好的团队合作精神和责任心，能够承受较强的工作压力；
    9. 熟悉Linux/Unix者优先考虑；
  • Andriod/Ios开发工程师 职位要求：
    1. 本职位面对0－2年工作经验者，本科学历，计算机及相关专业；
    2. 热爱编程，有扎实的基础，熟悉Java或者C++等任一门编程语言；
    3. 了解object-c/Android开发，并能较好地使用Xcode及Eclipse开发工具；
    4. 有IPhone或者Android经验更佳，但非必要条件；
    5. 对开发工作充满热情，具有较强的学习能力；
    6. 具有良好的团队合作精神和责任心，能够承受较强的工作压力；
  • 网页设计师 专业能力要求：
    1. 本科学历，美术及相关专业；
    2. 热爱设计、拥有宽广的行业（平面设计、UI设计、互联网）视野与时尚的审美标准；
    3. 熟悉网页制作（HTML+CSS），熟练使用photoshop；
    4. 强烈的创新欲望，关注细节，追求完美；
    5. 积极主动，良好的口头表达能力，学习能力强；
    6. 有3D设计经验更佳。
  • 软件测试工程师 专业能力要求：
    1. 计算机或相关专业本科以上学历；
    2. 熟悉软件测试理论、技术和工具，熟悉面向对象的测试；
    3. 具备较强的分析及解决问题的能力，良好的沟通协调能力和文档编写能力，以及较强的学习能力；
    4. 具有Windows、Linux操作系统管理以及Oracle、SQL-Server等数据库开发与管理经验；
    5. 有JAVA开发背景者优先考虑，熟悉报表的优先；
  • JAVA系统维护工程师 专业能力要求：
    1. 了解java基本知识及常用的struts/struts2,spring,hibernate框架；
    2. 熟悉linux基本操作；
    3. 有良好的沟通，表达能力，具有良好的团队合作精神和责任心；
    4. 认真负责，能够承受较强的工作压力；
    5. 有维护经验优先；
