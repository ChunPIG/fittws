数据生产过程管理平台

信息是现代企业的重要资源，是企业运用科学管理、决策分析的基础。目前，大多数企业花费大量的资金和时间来构建联机事务处理OLTP的业务系统和办公自动化系统，用来记录事务处理的各种相关数据。据统计，数据量每2～3年时间就会成倍增长，这些数据蕴含着巨大的商业价值，而企业所关注的通常只占在总数据量的2％～4％左右。因此，企业仍然没有最大化地利用已存在的数据资源，以致于浪费了更多的时间和资金，也失去制定关键商业决策的最佳契机。于是，企业面临如何通过各种技术手段，并把数据转换为信息、知识的问题。而ETL则解决这个问题的主要技术，通过ETL将数据整合和转化成知识，可以提高企业的核心竞争力。

Smart-ETL目前包括4个子产品：Designer Manager、Integration、Jobs Manager、DataQuality Manager。

Designer Manager

允许你通过图形界面来设计ETL转换流程（Transformation），支持拖拉的可视化图形操作，让业务管理人员不需要具备专业的IT水平都可以快速上手使用，可以更专注于数据业务层面。

Integration

集中运行管理由 Designer 设计的ETL流程 ，支持批量多任务并发处理工作，智能监控任务的运行状态。实现一次开发随处部署，提高开发的工作效率。具有高可用、高复用特性的集成平台。

Jobs Manager

任务管理器。 任务通过允许每个转换，任务，脚本等等，更有利于自动化更新数据仓库的复杂工作。任务通过允许每个转换，任务，脚本等等。任务将会被检查是否正确地运行。

DataQuality Manager

能够查找并解决令企业付出数百万元代价的数据质量问题。它提供了强劲的闭合循环数据探查、清洗、匹配、报告和跟踪功能。一套基于角色的工具让业内经理、业务分析人员和数据管理员能够与IT开发人员协作，保持数据的清洁性。IT 部门使用单一开发环境来管理数据质量，而所有人员和部门共享数据质量规则和工具。这一合作使得工作效率持续上升，成本持续降低。它提供了进入点数据验证、持续跟踪和异常管理，同时集中数据质量规则以支持任何应用程序。它同时还通过域觉察、预定规则和参考数据提供了高度精确的全局身份匹配和全局地址清洗。

Smart-ETL 的优势：

  • 兼容各类主流数据接口，彻底解决数据孤岛问题。

  • ETL任务并行处理，提供异乎寻常的任务执行速度。

  • 强大的数据联邦功能，支持跨异构库的数据关联。

  • 简单直观的图形操作界面，用户只需鼠标拖拽动作即可轻松完成ETL规则定义。

  • 提供工作流调度策略，自由定义任务的触发条件及执行路径，控制多任务的顺序或并发运行。

  • 集成脚本开发环境，支持语法加亮、断点调试、单步运行、变量监视等，充分满足二次开发需要。

  • 图形设计视图与脚本视图的自由切换，用户不仅可以享用快速开发带来的便利，同时可以满足用户查看、调试ETL细节的需求。

  • 多级数据安全策略，充分保证了ETL环节的数据安全。
